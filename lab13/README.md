# База данных аренды

Это веб-приложение для управления базой данных аренды, разработанное на PHP с использованием библиотеки FastRoute.

## Описание

Система управления арендой содержит три основные таблицы:
- **Арендаторы** (tenants): код арендатора, фамилия
- **Недвижимость** (rental_properties): код недвижимости, тип, ежемесячная арендная плата
- **Информация об аренде** (rental_info): дата начала аренды, срок аренды (в месяцах)

## Функциональность

### Основные возможности:
- Добавление, редактирование и удаление арендаторов
- Добавление, редактирование и удаление недвижимости
- Добавление, редактирование и удаление информации об аренде

### Отчеты:
1. Список недвижимости определенного типа, отсортированный по алфавиту или по цене
2. Список арендаторов для каждой недвижимости с указанием количества аренд
3. Список недвижимости, которая никогда не сдавалась
4. Список недвижимости, сдававшейся более 3 раз
5. Список недвижимости, сдававшейся более 2 раз со сроком аренды более 1 года
6. Список недвижимости с количеством аренд и общей суммой аренды
7. Список арендаторов с количеством аренд и средним сроком аренды
8. Список арендованной недвижимости указанного типа за указанный год и квартал
9. Список арендаторов и количество различных недвижимостей, которые они арендуют
10. Обновленная арендная плата для определенных типов недвижимости (увеличенная на 12%)

## Технологии

- **PHP** - основной язык программирования
- **MySQL** - система управления базами данных
- **FastRoute** - библиотека для маршрутизации
- **HTML/CSS/JavaScript** - веб-интерфейс
- **Chart.js** - визуализация данных

## Установка

1. Установите зависимости:
   ```bash
   composer install
   ```

2. Настройте подключение к базе данных в файле `src/database.php`

3. Запустите скрипт настройки базы данных:
   ```bash
   php src/setup_database.php
   ```

4. Запустите веб-сервер:
   ```bash
   cd public
   php -S localhost:8000
   ```

## Использование

После запуска приложения доступен веб-интерфейс по адресу `http://localhost:8000/`

Из интерфейса можно:
- Просматривать все данные
- Добавлять новые записи
- Удалять существующие записи
- Генерировать различные отчеты

## Структура проекта

- `public/index.php` - точка входа в приложение
- `public/src/` - исходный код приложения
  - `database.php` - настройки подключения к базе данных
  - `models.php` - модель данных и запросы к базе
  - `controllers.php` - контроллеры приложения
  - `views.php` - представления
  - `setup_database.php` - скрипт настройки базы данных
- `public/style.css` - файл стилей
- `vendor/` - установленные зависимости через Composer
